# ML-Classification-Models

**Project:**                   Building, training and testing Machine Learning classification models.

**Consumer:**                  California Franchise Tax Board

**Program languages:**         Python

**Python libraries are used:** numpy, pandas, sklearn, matplotlib, seaborn, etc.

**Description:**               As part of the EDR2 project (California State Enterprise Data to Revenue – Phase 2), I was part of a team of ML developers working on models that predict taxpayer compliance based on their payment histories. The repository includes one of the files detailing the construction, training, and evaluation of these classification models.

 - Confusion Matrix.png - screenshot of model quality assessment settings
 - ML Models Evaluation.py - Python file for developing a classification ML model
